Задача: вычислить число пи методом Монте Карло (на CPU и на GPU с применением CUDA)
Входные данные: N - количество точек
Выходные данные: вычисленное значение числа пи + время вычисления


Была реализована программа для вычисления числа пи. Две последовательности случайных чисел от 0 до 1 X и Y генерируются на GPU. Так как реализуется опреация редукции, то операции атомарные. Используется shared memory для осуществления параллельных вычислений. Каждая нить добавляет элемент вектора в массив разделяемой памяти, в рамках блока происходит синхронизация, нулевая нить вычисляет количество точек лежащих в окружности из shared memory и атомарно увеличивает глобальную переменную суммы.



Результаты запусков приведены в файле experimentsPi.xlsx.
