Задача: реализовать алгоритм сложения элементов вектора
Входные данные: Вектор размером 1 000..1 000 000 значений.
Выходные данные: сумма элементов вектора + время вычисления


Была реализована программа для вычисления суммы элементов вектора. Так как реализуется опреация редукции, то операции атомарные. Используется shared memory для осуществления параллельных вычислений. Каждая нить добавляет элемент вектора в массив разделяемой памяти, в рамках блока происходит синхронизация, нулевая нить вычисляет сумму элементов массива из shared memory и атомарно добавляет вычисленное значение в глобальную переменную суммы.


Результаты запусков приведены в файле experimentsVectorSum.xlsx.
